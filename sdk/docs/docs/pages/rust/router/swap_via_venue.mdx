# swap_via_venue

Swap through an explicit venue. `swap_via_venue_and_wait` waits and decodes
in one call.

```rust
pub async fn swap_via_venue(&self, venue: Address, params: &SwapParams) -> Result<H256>
pub async fn swap_via_venue_and_wait(&self, venue: Address, params: &SwapParams) -> Result<SwapResult>
```

## Usage

```rust
use propamm_sdk::common::pamms::KIPSELI;

let hash = router.swap_via_venue(KIPSELI, &params).await?;
```

## Returns

`Result<H256>`

The transaction hash. If the venue fails to fill at execution time, the
router falls back to Uniswap V3 transparently — the decoded result's
`executed_venue` reports what actually filled.

## Parameters

### venue

- **Type:** `Address`

A whitelisted pAMM or the fallback router address; anything else reverts
with `UnknownVenue`.

### params

- **Type:** `&SwapParams`

See [`swap`](/rust/router/swap#parameters).
