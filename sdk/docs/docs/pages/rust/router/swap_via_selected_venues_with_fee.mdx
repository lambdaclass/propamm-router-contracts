# swap_via_selected_venues_with_fee

[`swap_via_selected_venues`](/rust/router/swap_via_selected_venues) plus a
frontend fee skimmed from the output token.
`swap_via_selected_venues_with_fee_and_wait` waits and decodes in one call.

```rust
pub async fn swap_via_selected_venues_with_fee(&self, venues: Vec<Address>, params: &SwapParams, fee: FrontendFee) -> Result<H256>
pub async fn swap_via_selected_venues_with_fee_and_wait(&self, venues: Vec<Address>, params: &SwapParams, fee: FrontendFee) -> Result<SwapResult>
```

## Usage

```rust
use propamm_sdk::common::pamms::{BEBOP, FERMI};
use propamm_sdk::router::frontend_fee;

let hash = router
    .swap_via_selected_venues_with_fee(
        vec![FERMI, BEBOP],
        &params,
        frontend_fee(25, fee_recipient)?,
    )
    .await?;
```

## Returns

`Result<H256>`

The transaction hash.

## Parameters

### venues

- **Type:** `Vec<Address>`

The only venues considered for selection.

### params

- **Type:** [`&SwapParams`](/rust/types#swapparams)

See [`SwapParams`](/rust/types#swapparams); `amount_out_min` is the **net**
minimum after the fee.

### fee

- **Type:** [`FrontendFee`](/rust/types#frontendfee)

Build with [`frontend_fee(bps, recipient)`](/rust/router/frontend_fee).
