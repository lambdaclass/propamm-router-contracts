# swap_via_selected_venues

Re-quote only the supplied venues on-chain and swap through the best one;
routes to the Uniswap V3 fallback when none can be priced.
`swap_via_selected_venues_and_wait` waits and decodes in one call.

```rust
pub async fn swap_via_selected_venues(&self, venues: Vec<Address>, params: &SwapParams) -> Result<H256>
pub async fn swap_via_selected_venues_and_wait(&self, venues: Vec<Address>, params: &SwapParams) -> Result<SwapResult>
```

## Usage

```rust
use propamm_sdk::common::pamms::{BEBOP, FERMI};

let hash = router
    .swap_via_selected_venues(vec![FERMI, BEBOP], &params)
    .await?;
```

## Returns

`Result<H256>`

The transaction hash.

## Parameters

### venues

- **Type:** `Vec<Address>`

The only venues considered for selection. The Uniswap fallback still applies
at execution time as a safety net.

### params

- **Type:** `&SwapParams`

See [`swap`](/rust/router/swap#parameters).
