# swap_via_venue_with_fee

[`swap_via_venue`](/rust/router/swap_via_venue) plus a frontend fee skimmed
from the output token. `swap_via_venue_with_fee_and_wait` waits and decodes
in one call.

## Usage

```rust
use propamm_sdk::common::pamms::FERMI;
use propamm_sdk::router::frontend_fee;

let hash = router
    .swap_via_venue_with_fee(FERMI, &params, frontend_fee(25, fee_recipient)?)
    .await?;
```

## Returns

`Result<H256>`

The transaction hash.

## Parameters

### venue

- **Type:** `Address`

A whitelisted pAMM or the fallback router address.

### params

- **Type:** `&SwapParams`

See [`swap`](/rust/router/swap#parameters); `amount_out_min` is the **net**
minimum after the fee.

### fee

- **Type:** `FrontendFee`

Build with [`frontend_fee(bps, recipient)`](/rust/router/frontend_fee).
