# quote_uniswap_v3

Quote the Uniswap V3 fallback route directly, at the pair's resolved fee
tier. Never applies state overrides — the fallback quoter only reads live
on-chain pool state.

## Usage

```rust
use propamm_sdk::common::helpers::parse_ether;
use propamm_sdk::common::tokens::{USDC, WETH};

let amount_out = router.quote_uniswap_v3(WETH, USDC, parse_ether("1")?).await?;
// 1645288352 (1645.28 USDC)
```

## Returns

`Result<U256>`

The output amount the Uniswap V3 swap would produce.
