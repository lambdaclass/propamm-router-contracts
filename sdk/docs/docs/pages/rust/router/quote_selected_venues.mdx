# quote_selected_venues

Best quote among a caller-supplied set of venues.

```rust
pub async fn quote_selected_venues(&self, venues: Vec<Address>, token_in: Address, token_out: Address, amount_in: U256) -> Result<Quote>
pub async fn quote_selected_venues_with(&self, venues: Vec<Address>, token_in: Address, token_out: Address, amount_in: U256, opts: &QuoteOptions) -> Result<Quote>
```

## Usage

```rust
use propamm_sdk::common::pamms::{BEBOP, FERMI};

let quote = router
    .quote_selected_venues(vec![FERMI, BEBOP], USDC, WETH, amount_in)
    .await?;
```

## Returns

`Result<Quote>`

The best output amount among `venues` and the venue that produced it. Venues
that revert while quoting are skipped; when none can be priced the router
falls back to the Uniswap V3 quote.

## Parameters

### venues

- **Type:** `Vec<Address>`

The only venues considered.

### token_in / token_out / amount_in

See [`quote`](/rust/router/quote#parameters).

Same, plus `opts: &QuoteOptions` — see
