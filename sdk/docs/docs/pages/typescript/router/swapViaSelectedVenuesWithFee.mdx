# swapViaSelectedVenuesWithFee

[`swapViaSelectedVenues`](/typescript/router/swapViaSelectedVenues) plus a
frontend fee skimmed from the output token.
`swapViaSelectedVenuesWithFeeAndWait` waits and decodes in one call.

```ts
swapViaSelectedVenuesWithFee(venues: Address[], params: SwapParams, fee: FrontendFee): Promise<Hash>
swapViaSelectedVenuesWithFeeAndWait(venues: Address[], params: SwapParams, fee: FrontendFee): Promise<SwapResult>
```

## Usage

```ts
import { frontendFee } from "@propamm/sdk/router";
import { PAMMS } from "@propamm/sdk/common/pamms";

const hash = await router.swapViaSelectedVenuesWithFee(
  [PAMMS.fermi, PAMMS.bebop],
  params,
  frontendFee(25, "0x..."),
);
```

## Returns

`Hash`

The transaction hash.

## Parameters

### venues

- **Type:** `Address[]`

The only venues considered for selection.

### params

- **Type:** `SwapParams`

See [`swap`](/typescript/router/swap#parameters); `amountOutMin` is the
**net** minimum after the fee.

### fee

- **Type:** `FrontendFee`

Build with [`frontendFee(bps, recipient)`](/typescript/router/frontendFee).
