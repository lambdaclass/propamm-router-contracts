# swapViaVenue

Swap through an explicit venue. `swapViaVenueAndWait` waits and decodes in
one call.

```ts
swapViaVenue(venue: Address, params: SwapParams): Promise<Hash>
swapViaVenueAndWait(venue: Address, params: SwapParams): Promise<SwapResult>
```

## Usage

:::code-group

```ts [example.ts]
import { PAMMS } from "@propamm/sdk/common/pamms";
import { router } from "./config";

const hash = await router.swapViaVenue(PAMMS.kipseli, params);
```

```ts [config.ts]
import { ContractClient } from "@propamm/sdk/client";
import { PropAmmRouter } from "@propamm/sdk/router";
import { mainnet } from "@propamm/sdk/common/chains";
import { privateKeyToAccount } from "@propamm/sdk/common/accounts";

export const client = new ContractClient({
  rpcUrl: "https://...",
  chain: mainnet,
  account: privateKeyToAccount("0x..."),
});
export const router = new PropAmmRouter(client, "0x...");
```

:::

## Returns

`Hash`

The transaction hash. If the venue fails to fill at execution time, the
router falls back to Uniswap V3 transparently — the decoded result's
`executedVenue` reports what actually filled.

## Parameters

### venue

- **Type:** `Address`

A whitelisted pAMM or the fallback router address; anything else reverts
with `UnknownVenue`.

### params

- **Type:** `SwapParams`

See [`swap`](/typescript/router/swap#parameters).
