# quoteUniswapV3

Quote the Uniswap V3 fallback route directly, at the pair's resolved fee
tier. Never applies state overrides — the fallback quoter only reads live
on-chain pool state.

```ts
quoteUniswapV3(tokenIn: Address, tokenOut: Address, amountIn: bigint): Promise<bigint>
```

## Usage

:::code-group

```ts [example.ts]
import { parseEther } from "@propamm/sdk/common/helpers";
import { USDC, WETH } from "@propamm/sdk/common/tokens";
import { router } from "./config";

const amountOut = await router.quoteUniswapV3(WETH, USDC, parseEther("1"));
// 1645288352n (1645.28 USDC)
```

```ts [config.ts]
import { ContractClient } from "@propamm/sdk/client";
import { PropAmmRouter } from "@propamm/sdk/router";
import { mainnet } from "@propamm/sdk/common/chains";

export const client = new ContractClient({
  rpcUrl: "https://...",
  chain: mainnet,
});
export const router = new PropAmmRouter(client, "0x...");
```

:::

## Returns

`bigint`

The output amount the Uniswap V3 swap would produce.

## Parameters

### tokenIn

- **Type:** `Address`

Token being sold, or `ETH_SENTINEL` for native ETH.

### tokenOut

- **Type:** `Address`

Token being bought, or `ETH_SENTINEL` for native ETH.

### amountIn

- **Type:** `bigint`

Exact input amount, in atomic units.
