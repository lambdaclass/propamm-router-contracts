# swapViaVenueWithFee

[`swapViaVenue`](/typescript/router/swapViaVenue) plus a frontend fee
skimmed from the output token. `swapViaVenueWithFeeAndWait` waits and
decodes in one call.

## Usage

```ts
import { frontendFee } from "@propamm/sdk/router";
import { PAMMS } from "@propamm/sdk/common/pamms";

const hash = await router.swapViaVenueWithFee(
  PAMMS.fermi,
  params,
  frontendFee(25, "0x..."),
);
```

## Returns

`Hash`

The transaction hash.

## Parameters

### venue

- **Type:** `Address`

A whitelisted pAMM or the fallback router address.

### params

- **Type:** `SwapParams`

See [`swap`](/typescript/router/swap#parameters); `amountOutMin` is the
**net** minimum after the fee.

### fee

- **Type:** `FrontendFee`

Build with [`frontendFee(bps, recipient)`](/typescript/router/frontendFee).
