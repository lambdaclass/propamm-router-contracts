# Installation

The Python SDK is published as `propamm-sdk`, built on
[web3.py](https://web3py.readthedocs.io/). It requires Python 3.10+.

::: code-group

```bash [pip]
pip install propamm-sdk
```

```bash [uv]
uv add propamm-sdk
```

:::

The SDK is async — call it from within an `asyncio` event loop.
