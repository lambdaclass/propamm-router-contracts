# swapViaSelectedVenues

Re-quote only the supplied venues on-chain and swap through the best one;
routes to the Uniswap V3 fallback when none can be priced.
`swapViaSelectedVenuesAndWait` waits and decodes in one call.

```ts
swapViaSelectedVenues(venues: Address[], params: SwapParams): Promise<Hash>
swapViaSelectedVenuesAndWait(venues: Address[], params: SwapParams): Promise<SwapResult>
```

## Usage

```ts
import { PAMMS } from "@propamm/sdk/common/pamms";

const hash = await router.swapViaSelectedVenues( // [!code focus]
  [PAMMS.fermi, PAMMS.bebop], // [!code focus]
  params, // [!code focus]
); // [!code focus]
```

## Returns

`Hash`

The transaction hash.

## Parameters

### venues

- **Type:** `Address[]`

The only venues considered for selection. The Uniswap fallback still applies
at execution time as a safety net.

### params

- **Type:** [`SwapParams`](/typescript/types#swapparams)

See [`SwapParams`](/typescript/types#swapparams).
