# swapViaVenueWithFee

[`swapViaVenue`](/typescript/router/swapViaVenue) plus a frontend fee
skimmed from the output token. `swapViaVenueWithFeeAndWait` waits and
decodes in one call.

```ts
swapViaVenueWithFee(venue: Address, params: SwapParams, fee: FrontendFee): Promise<Hash>
swapViaVenueWithFeeAndWait(venue: Address, params: SwapParams, fee: FrontendFee): Promise<SwapResult>
```

## Usage

```ts
import { frontendFee } from "@propamm/sdk/router";
import { PAMMS } from "@propamm/sdk/common/pamms";

const hash = await router.swapViaVenueWithFee( // [!code focus]
  PAMMS.fermi, // [!code focus]
  params, // [!code focus]
  frontendFee(25, "0x..."), // [!code focus]
); // [!code focus]
```

## Returns

`Hash`

The transaction hash.

## Parameters

### venue

- **Type:** `Address`

A whitelisted pAMM or the fallback router address.

### params

- **Type:** [`SwapParams`](/typescript/types#swapparams)

See [`SwapParams`](/typescript/types#swapparams); `amountOutMin` is the
**net** minimum after the fee.

### fee

- **Type:** [`FrontendFee`](/typescript/types#frontendfee)

Build with [`frontendFee(bps, recipient)`](/typescript/router/frontendFee).
